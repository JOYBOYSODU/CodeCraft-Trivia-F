HTML stands for HyperText Markup Language. Think of it as the skeleton of every website! Just like how your body has bones to give it structure, websites use HTML to create their structure.

Every HTML document needs these basic parts:

<!DOCTYPE html> Tells the browser this is HTML5
<html> The root element of your webpage
<head> Contains information about your page
<body> Contains the visible content
<!DOCTYPE html> <html> <head> <title>My First Website</title> </head> <body> <h1>Hello World!</h1> </body> </html>
